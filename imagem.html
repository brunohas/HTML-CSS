IMAGENS NO SITE

Para pesquisar imagens, pode ser pelo google imagens mesmo, porém terá que escolher na opção que pesquise imagens sem direito autoral 'Marcadas para reutilização com modificação' ou 'Marcadas para reutilização'. Outras formas de pesquisar é com os sites: Unsplash (pesquisar em inglês), Pexels (pesquisar em inglês) ou Freepik.

Normalmente essas imagens são muito pesadas, por isso, editamos as fotos no GIMP, para editar abrimos a imagem com o GIMP → Imagem → Redimencionar imagem → Mudar largura entre 100 (pequeno) e 1500 (grande) e clicar em TAB para recalcular a altura, depois exportamos a imagem para a pasta com o código fonte. Podemos adicionar a imagem de 3 formas: arquivo da mesma pasta, arquivo de sub-pastas, e carregando direto do site (não recomendado pois não conseguimos editar a imagem e ela fica muito pesada).

Mesma pasta

Para colocar imagens no site, utilizamos a tag <img>, irá abrir 2 parâmetros: src=" " para colocar o nome do arquivo e alt=" " para identificar o que é a imagem.
Ex: 


                    <img src="foto1.jpg" alt="Skyline R34">

Sub-pasta

Para colocar uma imagem no site utilizando sub-pastas, também utilizamos a tag <img>, porém na hora de preencher os parâmetros, precisamos colocar o nome da pasta em que esta a imagem separado com uma barra (/) o nome do arquivo.
Ex: 


                    <img src="imagens/foto2.jpg" alt="Supra mk4">

Direto do site

Para colocar uma imagem direto do site, também utilizamos a tag <img>, e no lugar onde colocamos o nome do arquivo, para colocar uma imagem direto do site, colocamos a URL da imagem no src=" " (não recomendado pois não conseguimos editar a imagem, fica muito pesada). 


        <img src="https://media.istockphoto.com/id/521807412/pt/foto/nissan-silvia.jpg?s=612x612&w=0&k=20&c=80pKhwG1
        nw5x0QLRRQMUVuJlUaLqAjv9SOC5b9XAZnqU=" alt="Silvia S15">

ICON

No site Favicon conseguimos criar um icon para deixar no site (imagem que fica ao lado do título da página). Ou também podemos procurar por um icon já existente no site Icon Archive.

Para colocar um icon em um site, abrimos a tag <link:favicon> dentro do <head>. Depois irá abrir 3 parâmetros rel="shortcut icon" (não muda nada), href="favicon.ico" (colocar mesmo nome do arquivo do icon) type="image/x-icon"> (não muda nada).

Exemplo: 


    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Icon</title>
        <link rel="shortcut icon" href="imagens/icon.png" type="image/x-icon">
    </head>
    <body>

    </body>

IMAGENS DE FUNDO

Para colocar uma imagem de fundo, utilizamos o parâmetro background-image: url(); dentro da tag de estilo, colocar o nome do arquivo na url. Também conseguimos mudar o tamanho da imagem com o parâmetro background-size:, conseguimos mudar se a imagem se repete ou não, usar o parâmetro background-repeat:, conseguimos mudar o posicionamento da imagem, usar o parâmetro background-positon:, e conseguimos mudar o vinculo da imagem, utilizar o parâmetro background-attachment: (fixed = fixar a imagem no fundo / scroll = rolar a imagem junto com o conteúdo).

Sabemos que para fazer o alinhamento horizontal de uma imagem, utilizamos o parâmetro margin: auto;, agora para fazer um alinhamento vertical, adicionamos o parâmetro position: relative; na box de fundo e position: absolute; na box da frente. Com o position: absolute; conseguimos colocar outros parâmetros (top, left, bottom, right) para mexer a box da frente em relação a box do fundo, utilizar (left: 50%; top: 50%;), logo após, colocar o parâmetro transform: translate() (aplica uma alteração de transformação [translate() = mexe no componente]), colocar (-50%, -50%).
Ex:

Alinhamento Vertical

position: relative; para a box do fundo e position: absolute; + top: 50%; + left: 50%; + transform: translate(-50%, -50%) na box da frente.
Ex: 


                    <style>
                        body {
                            position: relative;
                        }

                        main > img {
                            position: absolute;
                            left: 50%;
                            top: 50%;
                            transform: translate(-50%, -50%);
                        }
                    </style>

Efeito Parallax

Um efeito Parallax é uma técnica de design que cria a ilusão de profundidade e movimento em um site, dando a impressão de um ambiente em 3D. Clique aqui para ir na página com um efeito Parallax (abrir os arquivos para enterder melhor). Para criar um efeito Parallax: criamos uma tag <section> e damos uma sub-classe class=" " e damos um nome para identificar que terá uma imagem de fundo e outra sub-classe id=" " se tiver mais do que uma imagem, para que no CSS coloque uma imagem para cada "id". Depois adicionar background-image: url() para adicionar a imagem, background-position: para definir a posição da imagem, background-repeat: para definir se a imagem se repete ou não, background-size: cover; para a imagem de fundo cubrir todo o contêiner sem distorção (indireita a imagem), e background-attachment: fixed; para fixar a imagem no fundo.
Ex: 


                    <style>
                        section.img {    !- todas as class="img" -!
                            background-attachment: fixed;
                            background-position: center center;
                            background-size: cover;
                        }

                        section#img1 {   !- imagem para primeiro id -!
                            background-image: url(imagem1);
                        }

                        section#img2 {   !- imagem para o segundo id -!
                            background-image: url(imagem2);
                        }
                    </style>

IMAGEM DINÂMICA

Para fazer uma uma imagem dinâmica, utilizamos a tag <picture>, pois com ela conseguimos "trocar" a imagem para que caiba na tela no usuário. Abrimos a tag <picture> e dentro dessa tag abrimos uma tag <img> para a imagem principal, depois abrimos outra tag <source:media:type> para cada imagem substituta (deve ser a mesma imagem, porém com a resolução diferente). Depois irá abrir 3 parâmetros media="(min-width: )" (trocar min por max), srcset=" " (colocar nome do arquivo), e type="image/" (colocar o tipo do arquivo). A ordem é como se fosse uma pirâmide invertida, colocar a tag <img> em último e acima dela colocar as tag <source:media:type> de acordo com a quantidade de fotos e de tamanho (maior em último (tag <img>) / média no meio (tag <source:media:type>) / pequena acima do outro source (tag <source:media:type>)).

Na hora que abrir a tag <source:media.type> mudar a opção (min-width: ) para (max-width: ) e sempre colocar 20px amais do que a foto que você quer colocar, exemplo: a foto G tem tamanho de 1000px → no source da foto M coloco o max-width: 1020px para que se a tela não suportar a imagem principal = foto G abrir a foto M.

Para entender melhor clique aqui para ir na página de exemplo.

Exemplo:


    <picture>
        <source media="(max-width: 720px)" srcset="imagens/foto-p.png" type="image/png">  !-Se a tela for menor que 720px-!
        <source media="(max-width: 1020px)" srcset="imagens/foto-m.png" type="image/png"> !-Se a tela for menor que 1050px-!
        <img src="imagens/foto-g.png" alt="Imagem flexível">                              !- Por padrão irá abrir essa imagem -!
    </picture>
