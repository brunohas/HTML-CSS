LISTAS

Conseguimos fazer 2 tipos de listas, listas ordenadas (utilizar a tag <ol>) e listas não ordenadas (utilizar a tag <ul>). Para colocar um item em alguma dessas listas, utilizamos a tag <li>.

Listas Ordenadas

Como o nome já diz, é uma lista ordenada, porém conseguimos adicionar um parâmetro start=" " dentro da tag <ol> para conseguir escolher em qual número ou letra a lista irá começar. Também conseguimos mudar o ícone que fica na lista utilizando o parâmetro type=" ".

  1. type="1" → lista númerada
  1. type="A" → lista alfabética maiúscula
  1. type="a" → lista alfabética minúscula
  1. type="I" → lista em número romano maiúscula
  1. type="i" → lista em número romano minúscula

Listas não Ordenadas

Para abrir uma lista não ordenada, utilizamos a tag <ul>, e para cada item da lista usar a tag <li>. Também conseguimos adicionar o parâmetro type=" " para mudar a "bolinha" da lista.

  • type="disc" → bolinha preta
  • type="circle" → bolinha branca
  • type="square" → quadradinho preto

Em uma lista não ordenada conseguimos colocar um emoji no lugar da "bolinha", dentro da tag de estilo utilizar o parâmetro list-style-type: '\XXXX', no lugar desses X colocar o número do emoji ('\00A0' faz um espaço em branco). E com o parâmetro list-style-position: conseguimos colocar a "bolinha" dentro (inside) ou fora (outside).

Listas Compostas

Para fazer uma lista composta, basta se atentar a identação, nesse exemplo vou abrir uma lista ordenada dentro de uma lista não ordenada. 


            <ul>                                           !- abrir lista não ordenada -!
                <li>Nintendo</li>                          !- item da lista não ordenada -!
                <ol>                                       !- abrir lista ordenada -!
                    <li>Pokemon Fire Red</li>              !- item da lista ordenada -!
                    <li>Super Mario Kart</li>              !- item da lista ordenda -!
                    <li>Kurby Super Stars</li>             !- item da lista ordenada -!
                </ol>                                      !- fechamento a lista ordenada -!
                <li>XBOX</li>                              !- item da lista não ordenada -!
                <ol>                                       !- abrir lista ordeanada -!
                    <li>Gears Of War 2</li>                !- item da lista ordenada -!
                    <li>Forza</li>                         !- item da lista ordenada -!
                    <li>Prototype</li>                     !- item da lista ordenada -!
                </ol>                                      !- fechamento da lista ordeanda -!
                <li>PlayStation</li>                       !- item da lista não ordenada -!
                <ol>                                       !- abrir lista ordenada -!
                    <li>Call of Duty</li>                  !- item da lista ordenada -!
                    <li>Grand Theft Auto</li>              !- item da lista ordenada -!
                    <li>Apex Legends</li>                  !- item da lista ordeanda -!
                </ol>                                      !- fechamento da lista ordenada -!
            </ul>                                          !- fechamento lista não ordenada -!

  • Nintendo
    1. Pokemon Fire Red
    2. Super Mario Kart
    3. Kurby Super Stars
  • XBOX
    1. Gears Of War 2
    2. Red Dead Redemption
    3. Prototype
  • PlayStation
    1. Call of Duty
    2. Grand Theft Auto
    3. Apex Legends

Lista de Definições

Utilizar a tag <dl> para abrir a lista, tag <dt> para o item da lista, e a tag <dd> para a descrição do item.
Ex: 


            <dl>
                <dt>HTML</dt>
                <dd>Linguagem de marcação para a criação do conteúdo de um site.</dd>
                <dt>CSS</dt>
                <dd>Linguagem de marcação para a criação do desing de um site.</dd>
                <dt>JavaScript</dt>
                <dd>Linguagem de programação para criação de interatividade de um site.</dd>
            </dl>

Que no caso ficaria assim:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do desing de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site.

TABELAS

Para criar tabelas em HTML, utilizamos a tag <table> para abrir a tabela, <caption> para o título / legenda da tabela junto com as tags <tr> (table row = linha da tabela), <td> (table data = dado da tabela (número)), e <th> (table head = cabeçalho da tabela (alfabética)). Quando criamos uma tabela e acrescentamos a tag de estilo border: 0px solid black; para criar linhas em volta de cada dado, porém elas ficam com um pequeno espaço entre elas, para tirar esse espaço, utilizamos a tag de estilo border-collapse: collapse;, e para ter um espaço utiliza-se a tag de estilo border-collapse: separate;.

Tabelas Simples


        <table>                                          |          <style>                                            
        <caption>Tabela simples</caption>                |              table {             
            <tr>                                         |                  width: 500px;                                            
                <td>A1</td>                              |                  border-collapse: collapse;            
                <td>B1</td>                              |                  margin: auto;                 
                <td>C1</td>                              |               }                 
            </tr>                                        |               caption {                       
            <tr>                                         |                  border: 1px solid black;                
                <td>A2</td>                              |               }                 
                <td>B2</td>                              |               td {                     
                <td>C2</td>                              |                  border: 1px solid black;                              
            </tr>                                        |                  padding: 10px;                      
            <tr>                                         |                  text-align: center;                   
                <td>A3</td>                              |                  vertical-align: middle;                   
                <td>B3</td>                              |               }                      
                <td>C3</td>                              |           </style>                             
            </tr>                                        |

Que no caso ficaria assim:

Tabela Simples
A1 B1 C1
A2 B2 C2
A3 B3 C3

Para fazer tabelas mais complexas, ainda utilizamos a tag <table>, porém com mais algumas outras tags, <thead> para o cabeçalho da tabela, <tbody> para o corpo da tabela, <tfoot> para o rodapé da tabela. E dentro deles utilizamos as mesmas tags da tabela simples: <tr> (linha da tabela) <td> (dado da tabela) <th> (cabeçalho da tabela). 


                                        <table>
                                            <caption> </caption>
                                            <thead>
                                                <tr>
                                                    <th ou td> (qnts vezes necessário)
                                                </tr> 
                                            </thead>
                                            <tbody>
                                                <tr>
                                                    <th ou td> (qnts vezes necessário)
                                                </tr>
                                            </tbody>
                                            <tfoot>
                                                <tr>
                                                    <th ou td> (qnts vezes necessário)
                                                </tr>
                                            </tfoot>

Para deixar claro para os mecânismos de busca, que um título de uma tabela é relacionado com a coluna, acrescentar o parâmetro scope="col", e o título relacionado com a linha, utilizar o parâmetro scope="row". Isso não causa nenhum efeito visual, serve só para especificar o relacionamento entre o título e os dados.

Conseguimos fazer um efeito zebrado nas tabelas usando uma tag de estilo, usamos na tag <tr>, utilizar nth-child(even) ou nth-child(odd). Também podemos trocar o even (par) por 2n, e odd (ímpar) por 2n-1. Que no caso as tags de estilo ficariam assim: 


                    tbody > tr:nth-child(even) {                                  tbody > tr:nth-child(odd) {
                        background-color: lightgray;              OU                  background-color: lightgray;
                    }                                                             }

Para fazer um cabeçalho fixo em uma tabela, utilizamos as tags de estilo position: relative; na tag da tabela (table), e criamos outra tag de estilo (thead > tr > th) que indica que o <th> está dentro do <tr> que está dentro do <thead> e vai receber as seguintes tags de estilo: position: sticky; (colável) + top: 0; (colar na parte 0 "topo") + background-color: (msm cor do cabeçalho) (não é obrigatório). Que no caso ficariam assim: 


                    table{                                          thead > tr > th {
                        position: relative;          +                  position: sticky;
                    }                                                   top: 0;
                                                                        background-color: msm do cabeçalho;
                                                                    }
