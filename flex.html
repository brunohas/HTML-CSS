flex-flow

flex-flow: row wrap; (alinha esquerda p/ direita + quebra de linha qnd item flexível não puder mais ser compactado)

flex-flow: column wrap-reverse; (alinha única coluna de cima para baixo + quebra de linha ao contrária, para a linha acima)

flex-flow: column-reverse nowrap; (alinha única coluna de baixo para cima + não permite quebra de linha)
