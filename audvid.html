Áudio

Para reproduzir um áudio MP3 em um site, utilizamos a tag <audio>, irá abrir um parâmetro src=" " (colocar o nome do arquivo) e também é preciso colocar os parâmetros autoplay para tocar a música assim que carregar a página (não é obrigatório), controls para aparecer um "controle" para dar play/pause e aumentar ou diminuir o volume do áudio, loop para o áudio se repetir (usa-se normalmete quando o áudio é curto). 


    <audio src="midia/THE BOX MEDLEY FUNK 2.mp3" controls></audio>

Também existe outros tipos de formatos de áudio (WAV e OGG), ver se tem compatibilidade com o navegador. Outra maneira de reproduzir áudios também é com a tag <audio>, porém não é necessário o parâmetro que já vem incluso (pode apagar o src=" ") se quiser pode acrescentar o parâmetro preload=" " (auto = só considera que a página carregou depois de carregar todo o áudio (não recomendado) / metadata = só carrega o nome, data, duração do arquivo / none = só carrega quando o usuário apertar play no controle). Depois colocar a tag <source:src> dentro da tag <audio>, assim abrindo outro parâmetro type=" ", para áudios em MP3 colocar o type="audio/mpeg" / para áudios em WAV colocar o type="audio/wav" / para áudios em OGG colocar o type="OGG".

Exemplo:


    <audio controls>
        <source src="midia/THE BOX MEDLEY FUNK 2.mp3" type="audio/mpeg">  !- primeiro áudio a tentar carregar -!
        <source src="midia/THE BOX MEDLEY FUNK 2.wav" type="audio/wav">   !- se não carregar o primeiro, carrega o segundo -!
        <source src="midia/THE BOX MEDLEY FUNK 2.ogg" type="audio/ogg">   !- se não carregar o segundo, carrega o terceiro -!
        <p>
            Infelizmente seu navegador não consegue reproduzir áudio.     !- Só irá mostrar se não conseguir produzir o áudio -!
        </p>
    </audio>

VÍDEO

Os formatos mais utilizados em vídeos são o MP4 e M4V, existem outros como WebM (que é um arquivo bem pequeno, porém não tem muita qualidade de vídeo) e OGV (um formato que está intimamente ligado ao OGG, pode ser utilizado para armazenamento ou transmissão na web).

Para pesquisar vídeos, podemos também utilizar o site Pexels que é o mesmo para pesquisar imagens (sempre lembrar dos direitos autorais). Para converter/formatar um vídeo utilizamos o aplicativo HandBrake (só converte para MP4, M4V e WebM / para converter para OGV basta pesquisar no google "conversor ogv"), serve para converter o vídeo em algum outro tipo de arquivo para que o vídeo fique mais leve, ou para o vídeo ficar com uma resolução melhor. Não é indicado hospedar um arquivo muito no grande no site, pois o custo fica muito caro. Se for colocar algum arquivo muito pesado, é indicado colocar ele como um link externo.
