RESPONSIVIDADE

A responsividade de um site é a capacidade de se adaptar a diferentes tamanhos de tela. O objetivo é proporcionar uma boa experiência de visualização ao usuário, independentemente do dispositivo que esteja sendo utilizado. Para saber melhor a medida ou visualizar em tela de celular, utilizar a extensão do Google Chrome Window Resizer. Clique aqui para ir no meu primeiro projeto com responsividade.

Responsividade da tela

Para criar essa responsividade, utilizamos os parâmetros min-width: (largura mínima da tela) e max-width: (largura máxima da tela). Normalmente a largura mínima será de 320px (pois é a largura de alguns celulares mais antigos). 


                                            <style>
                                                * {
                                                    margin: 0px;
                                                    padding: 0px;
                                                }
                                                
                                                body, html {
                                                    heigth: 100vh;
                                                    min-width: none;
                                                    max-width: 100vw;
                                                }
                                            </style>

Responsividade da imagem

Para fazer a responsividade de uma imagem, adicionar a tag de estilo somente para a imagem (img {width: 100%;}) para que ela "quebre" a largura junto com a janela. 


                                            <style>
                                                img {
                                                    width: 100%;
                                                    display: block;
                                                    margin: auto;
                                                    max-width: largura da imagem;
                                                }

Ou também podemos utilizar a tecnica da imagem dinâmica mostrada na página Imagens.

Responsividade de vídeo

Para fazer a responsividade de um vídeo, envelopamos a tag de <video> ou <iframe> com uma tag <div>, e adicionamos um class=" " para identificar que é um vídeo. Depois criamos 2 parâmetros na tag de estilo, div. {} e div. > video {} ou div. > iframe {}, para colocar as seguintes tags: 


        <div class="video">                                  |    <div class="video">
            <video src=" "></video>                         ou        <iframe src=" " frameborder="0"></iframe>
        </div>                                               |    </div>

                    <style>
                        div.video {
                            padding-bottom: %; (50%)          !- parte preta dentro do vídeo -!
                            position: relative;               !- posição da tag <div> -!
                        }

                        div.video > video {
                            position: absolute;               !- posição do video dentro da tag <div>
                            top: %; (5%)                      !- distância do topo -!
                            left: %; (5%)                     !- distância da esquerda -!
                            width: %; (90%)                   !- largura do vídeo -!
                            heigth: %; (90%)                  !- altura do vídeo -!
                        }
                    </style>

                    Colocar as medidas em porcentagem (%), para que consiga fazer a responsividade.

Responsividade de tabela

Para fazer uma responsividade de uma tabela de um site, envelopamos com uma tag <div> todo o conteúdo que estiver dentro da tag <table> e acrescentamos class=" " para identificar que é uma tabela. Depois acrescentamos uma tag de estilo (overflow: → os dois eixos / overflow-y: → eixo vertical / overflow-x: → eixo horizontal) dentro da tag <div> para que a tabela "quebre" e não tenha uma barra de rolagem lateral. 


    <div class="tabela">                                       |    <style>
        <table>                                                |        div.tabela {
            <caption>Tabela Responsiva</caption>               |            overflow-x: auto;  !- barra de rolagem horizontal -!
            <tbody>                                            |            margin: auto;      !- tabela ficar no meio -!

                </tr>                                          |        table {
                <tr>                                           |            width: 600px;

Menu Responsivo

Para criar um menu responsivo, utilizamos o site do Google Icons para pegar um ícone, é utilizado igual o Google Fonts (escolher o ícone, copiar e colar o código do mesmo) W3. Depois criamos uma tag <span> e acrescentamos o parâmetro class="material-symbols-outlined" dentro da tag, e para fazer a interação com o menu, adicionamos o parâmetro onclick="clickMenu()" dentro da tag <span>. Depois criamos uma tag <menu> com uma lista de links para direcionar, e no final cria-se uma tag <script> para fazer a interação (JavaScript). 


    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Menu responsivo</title>
        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined
        :opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&icon_names=menu"/>    !- código copiado do Google Icons -!
    </head>
    <body>
        <header>
            <h1>Menu responsivo</h1>
            <span class="material-symbols-outlined" onclick="clickMenu()">menu</span>  !- código copiado do Google Icons -!
            <menu>
                <ul>
                    <li><a href=" ">Opc1</a></li>                              
                    <li><a href=" ">Opc2</a></li>                               !- links p/ direcionamento -!
                    <li><a href=" ">Opc3</a></li>
                </ul>
            </menu>
        </header>
        <main>

        </main>
